The goal here is to have a simple text field in the general tab of Woo's single-product.php so that when the store owner is typing up their wonderful new product, they can customize text that appears on that particular product's add to cart button.

== Description ==

The plugin is for WooCommerce store owners who simply need to change the "Add To Cart" text, and to give them a simple & easy way to do that.

== Installation ==

1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Place `<?php do_action('plugin_name_hook'); ?>` in your templates
